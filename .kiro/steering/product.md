# Product Overview

This is a static event landing page template for **Community Builder Days 2025**, a community-focused conference event.

## Purpose

Single-page promotional website for event marketing with sections for:

- Event hero/countdown
- Highlights and key features
- Speaker lineup
- Schedule/agenda
- Venue information
- FAQ
- Sponsors
- Call-to-action (registration)

## Content Management

All event data is centralized in `src/constants/` for easy customization:

- Event details (dates, location, registration links)
- Navigation items
- Speakers, schedule, FAQ, sponsors, highlights

This separation allows non-technical users to update event content without touching component code.
